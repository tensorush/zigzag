## :lizard: :trident: **zigzag**

[![CI][ci-shd]][ci-url]
[![LC][lc-shd]][lc-url]

### Multi-threaded CPU path tracer.

#### :rocket: Usage

```sh
git clone https://github.com/tensorush/zigzag.git
cd zigzag/
zig build exe
```
